Installation

Flash the official firmware first, then proceed with the custom firmware/theme.

I always import assets onto a fresh Fiio image and avoid modifying an already modified one. I do not recommend editing an already modified image unless you know exactly what you are doing.

I usually wait a few seconds after writing files and always use Windows “Safely Remove Hardware” before unplugging (even though the Echo Mini has a hardware eject option).
